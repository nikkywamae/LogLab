Each of the php files holds one element, or "page" of the code we wrote and uploaded on AwardSpace to make 
our prototype for the LogLab website. 
I thought it would be easier to upload all of the files to one place, which is why I chose the Github. 

The website is pretty simple, but just to restate the objectives outlined in the first milestone we submitted:
LogLab is a database that allows scientists and science lovers to connect. Similar to websites like PubMed
LogLab allows scientists to upload thier work and communicate with those interested. If not interested 
in talking one on one with an owner of an article, unregistered users are free to browse and consume! (They just cant message)
